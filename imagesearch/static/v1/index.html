API: Image Search Abstraction Layer

User Story: I can get the image URLs, alt text and page urls for a set of images relating to a given search string.

User Story: I can paginate through the responses by adding a ?offset=2 parameter to the URL.

User Story: I can get a list of the most recently submitted search strings.

Usage:
Get the image URLs in the JSON response for search "lolcats funny":
https://myprtfl.herokuapp.com/api/imagesearch/v1/lolcats%20funny

Get list of the most recently submitted search strings in the JSON response:
https://myprtfl.herokuapp.com/api/imagesearch/v1/is-stat/latest
